# 骨架屏

## 特性

## 用法

```
    Vue.use(Vskleton,{directiveName:'skleton'})
```

## 参数说明

| 参数          | 说明   | 类型   | 是否必填 | 可选值 | 默认值  |
| ------------- | ------ | ------ | -------- | ------ | ------- |
| directiveName | 指令名 | String | no       |        | skleton |

## demo

```
    <div v-skleton>lixiwang</div>
```
